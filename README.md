
# ScholaX

## Setup Instructions

### Prerequisites
- Node.js (v14 or higher)
- MySQL/XAMPP
- npm or yarn

### Setup
```bash
cd backend
npm install
cp .env.example .env  # Configure your database
npm run dev



# 1. clone the repository
git clone https://github.com/xveraaaa/ScholaX.git
cd ScholaX

# 2. install backend dependencies
cd backend
npm install

# 3. install frontend dependencies
cd ../frontend
npm install

# 4. run both
# backend (terminal 1)
cd backend
npm run dev

# frontend (terminal 2)
cd frontend
npm run dev
